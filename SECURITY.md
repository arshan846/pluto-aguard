# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Pluto AgentGuard, please report it responsibly.

**Email:** arpitha.dhanapathi@gmail.com

**What to include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

**Response time:** I aim to acknowledge reports within 48 hours.

**Scope:** This covers the `pluto-aguard` Python package and CLI tool. It does not cover the projects you scan with it — AgentGuard is a detection tool, not an enforcement runtime.

## Supported Versions

| Version | Supported |
|---|---|
| 0.9.x | ✅ Current |
| < 0.9 | ❌ Upgrade recommended |
